# Standard

Ensure all dependencies within this group have the specified semver range.

## `semverGroup.range`

<Pills required />

Which of the Supported Ranges this group should use.

```json title=".syncpackrc"
{
  "semverGroups": [
    {
      "range": "~",
      "dependencies": ["@alpha/**"],
      "packages": ["**"]
    }
  ]
}
```
