# Same Range

Relax syncpack to ensure that all versions have semver ranges which all satisfy
each other, instead of the default behaviour of them having to be identical.

:::info

- One of the possible values for the [`versionGroups`](../version-groups.mdx)
  configuration array.
- Learn more in our [guide to Version Groups](../../guide/version-groups.mdx).

:::

## `policy` "sameRange"

<Pills required />

Currently the only custom policy in syncpack is "sameRange", which is what
causes this version group to be used.

```json title="..."
{
  "versionGroups": [
    {
      "dependencies": ["**"],
      "packages": ["**"],
      "policy": "sameRange"
    }
  ]
}
```
