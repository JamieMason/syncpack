# `sortFirst` string[]

<Pills optional />

When using the `format` command, determines which fields within package.json
files should appear at the top, and in what order.

## Default Value

```json title=".syncpackrc"
{
  "sortFirst": ["name", "description", "version", "author"]
}
```

:::tip

Syncpack config files also support
[TypeScript IntelliSense](https://jamiemason.github.io/syncpack/config-file#typescript-intellisense).

:::
