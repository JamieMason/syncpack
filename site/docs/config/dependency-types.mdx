Areas where versions are usually defined can be toggled on and off via this property.

<details>
  <summary>Example: Only include <code>dependencies</code> and <code>devDependencies</code></summary>

**Only** `dependencies` and `devDependencies` will be inspected:

```json title=".syncpackrc"
{
  "dependencyTypes": ["dev", "prod"]
}
```

</details>

<details>
  <summary>Example: Exclude <code>dependencies</code> and <code>devDependencies</code></summary>

Everything **except** `dependencies` and `devDependencies` will be inspected:

```json title=".syncpackrc"
{
  "dependencyTypes": ["!dev", "!prod"]
}
```

</details>
