---
title: RefuseToSnapLocal
status: suspect
description: Local workspace package cannot be snapped to another package and must be handled differently
---

## When this happens

- ✘ [Local Instance](REF_GLOSSARY#local-instance) is in a [Snapped To](VERSION_GROUP_SNAPPED_TO) version group
- ✘ An Instance of this dependency was found in the snapped to package
- ✘ Misconfiguration: Syncpack refuses to change the version specifier of a [Local Instance](REF_GLOSSARY#local-instance)
