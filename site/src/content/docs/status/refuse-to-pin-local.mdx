## When this happens

- ✓ Instance is in a [Pinned](VERSION_GROUP_PINNED) version group
- ✘ [Local Instance](REF_GLOSSARY#local-instance) mismatches its pinned version group
- ✘ Misconfiguration: Syncpack refuses to change the version specifier of a [Local Instance](REF_GLOSSARY#local-instance)
