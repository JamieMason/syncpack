---
title: IsIdenticalToPin
status: valid
description: Dependency version is valid and matches the pinned version specified in its version group
---

## When this happens

- ✓ Instance is in a [Pinned](VERSION_GROUP_PINNED) version group
- ✓ Instance is identical to its pinned version group
- ✓ Instance matches its [With Range](SEMVER_GROUP_WITH_RANGE) semver group
