## When this happens

- ✓ Instance is in a [Snapped To](VERSION_GROUP_SNAPPED_TO) version group
- ✓ Instance is identical to a matching [snapTo](VERSION_GROUP_SNAPPED_TO#snapto) instance
- ✓ Instance matches its [With Range](SEMVER_GROUP_WITH_RANGE) semver group
