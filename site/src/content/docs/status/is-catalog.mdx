---
title: IsCatalog
status: valid
description: Dependency version is valid and uses the catalog protocol as required by its version group
---

## When this happens

- ✓ Instance is in a [Highest Semver](VERSION_GROUP_HIGHEST_SEMVER) or [Lowest Semver](VERSION_GROUP_LOWEST_SEMVER) version group
- ✓ Instance uses the `catalog:` protocol and wins out
