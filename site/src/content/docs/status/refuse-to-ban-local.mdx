## When this happens

- ✓ Instance is a [Local Instance](REF_GLOSSARY#local-instance)
- ✘ Instance is in a [Banned](VERSION_GROUP_BANNED) version group
- ✘ Misconfiguration: Syncpack refuses to change the version specifier of a [Local Instance](REF_GLOSSARY#local-instance)
