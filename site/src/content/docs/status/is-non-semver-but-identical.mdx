## When this happens

- ✓ Instance is in a [Highest Semver](VERSION_GROUP_HIGHEST_SEMVER) or [Lowest Semver](VERSION_GROUP_LOWEST_SEMVER) version group
- ! No Instances have [Simple Semver](REF_GLOSSARY#simple-semver) version specifiers
- ✓ Instance is identical to every other instance in its version group
