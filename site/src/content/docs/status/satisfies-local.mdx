---
title: SatisfiesLocal
status: valid
description: Dependency version is valid and satisfies the version range defined in its local workspace package
---

## When this happens

- ✓ Instance is in a [Highest Semver](VERSION_GROUP_HIGHEST_SEMVER) or [Lowest Semver](VERSION_GROUP_LOWEST_SEMVER) version group
- ✓ Instance matches the version of its [Local Instance](REF_GLOSSARY#local-instance)
- ✓ Instance matches its [With Range](SEMVER_GROUP_WITH_RANGE) semver group
- ! Considered a loose match we should highlight
