---
title: RefuseToCatalogLocal
status: suspect
description: Local workspace package cannot use the catalog protocol and must be excluded from the catalog group
---

## When this happens

- ✓ Instance is in a [Catalog](VERSION_GROUP_CATALOG) version group
- ✓ Instance is a [Local Instance](REF_GLOSSARY#local-instance) (its own `package.json` `/version` property)
- ✘ Local instances cannot use the `catalog:` protocol
- ! Reconfigure version groups to exclude local instances from the catalog group
