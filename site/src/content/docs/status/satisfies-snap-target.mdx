## When this happens

- ✓ Instance is in a [Snapped To](VERSION_GROUP_SNAPPED_TO) version group
- ✓ Instance has same [Semver Number](REF_GLOSSARY#semver-number) as matching [snapTo](VERSION_GROUP_SNAPPED_TO#snapto) instance
- ✓ Instance matches its [With Range](SEMVER_GROUP_WITH_RANGE) semver group
- ✓ Range preferred by semver group satisfies the matching [snapTo](VERSION_GROUP_SNAPPED_TO#snapto) instance
- ! Considered a loose match we should highlight
