## When this happens

- ✓ Instance is in a [Catalog](VERSION_GROUP_CATALOG) version group
- ✓ [MissingFromCatalog](STATUS_MISSING_FROM_CATALOG) applies to multiple instances of the same dependency
- ✘ Their specifiers differ AND at least one is non-semver
- ? Syncpack cannot pick which specifier to enshrine in the catalog
