Export dependency data from your monorepo as JSON. Outputs one line per dependency instance with version, location, and status information. Use with tools like [`jq`](https://jqlang.org/) to analyse, filter, or pipe into other programs.

:::tip
The [fix](COMMAND_FIX) and [format](COMMAND_FORMAT) commands also support `--reporter json` for machine-readable output of their results. Use `syncpack fix --reporter json` for fixable version mismatches or `syncpack format --reporter json` for formatting issues.
:::

## Examples

```bash
# Output all dependencies as JSON
syncpack json
# Output only AWS SDK dependencies
syncpack json --dependencies '@aws-sdk/**'
# Count dependencies by type
syncpack json | jq -r '.dependencyType' | sort | uniq -c
# See more examples
syncpack json --help
# See a short summary of options
syncpack json -h
```
