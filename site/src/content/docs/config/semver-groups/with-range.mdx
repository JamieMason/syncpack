## Configuration

### range

<RequiredPill />

Which of the Supported Ranges this group should use.

```json title=".syncpackrc"
{
  "semverGroups": [
    {
      "range": "~",
      "dependencies": ["@alpha/**"],
      "packages": ["**"]
    }
  ]
}
```
