Have syncpack ignore the semver ranges of these dependencies completely.

## Configuration

### isIgnored <Badge text="Required" variant="danger" />

```json title="Hide dependencies from syncpack"
{
  "semverGroups": [
    {
      "packages": ["oops-moment", "workaround"],
      "dependencies": ["**"],
      "isIgnored": true
    }
  ]
}
```
