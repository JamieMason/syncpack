Fix mismatched versions in this group by choosing the lowest/oldest semver version in use.

## Examples

```json title="Choose the lowest valid semver version when fixing mismatches"
{
  "versionGroups": [
    {
      "packages": ["**"],
      "dependencies": ["**"],
      "dependencyTypes": ["**"],
      "preferVersion": "lowestSemver"
    }
  ]
}
```
