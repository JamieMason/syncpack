## Configuration

<RequiredPill>
  {/**/}
  ### isBanned
</RequiredPill>

This property activates this behaviour for a given Version Group.

```json title=".syncpackrc"
{
  "versionGroups": [
    {
      "dependencies": ["never-gonna"],
      "isBanned": true
    }
  ]
}
```
