## Configuration

<RequiredPill>
  {/**/}
  ### isIgnored
</RequiredPill>

This property activates this behaviour for a given Version Group.

```json title=".syncpackrc"
{
  "versionGroups": [
    {
      "dependencies": ["keep-walking"],
      "isIgnored": true
    }
  ]
}
```
