Patterns supported by [glob](HREF_GLOB) to find package.json files you want to manage with syncpack.

## Default Value

Defaulted to match most Projects using Lerna or Yarn Workspaces

```json title=".syncpackrc"
{
  "source": ["package.json", "packages/*/package.json"]
}
```
