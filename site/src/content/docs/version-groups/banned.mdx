Prevent dependencies from being used anywhere in your monorepo, or only in specific packages. Common cases are blocking legacy packages, conflicting libraries, or packages that violate your security or licensing policy. When syncpack finds a banned dependency, it flags it as an issue that must be removed.

## Configuration

### isBanned <Badge text="Required" variant="danger" />

This property activates this behaviour for a given Version Group.

```json title=".syncpackrc.json"
{
  "versionGroups": [
    {
      "dependencies": ["never-gonna"],
      "isBanned": true
    }
  ]
}
```
