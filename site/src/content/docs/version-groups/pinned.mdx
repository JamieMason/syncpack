Pin the version of all dependencies in this group to match a specific version you've defined.

## Configuration

### pinVersion <Badge text="Required" variant="danger" />

The version specifier you would like to use, this can be anything supported by a package manager.

```json title=".syncpackrc.json"
{
  "versionGroups": [
    {
      "dependencies": ["@types/node"],
      "pinVersion": "18.14.2"
    }
  ]
}
```
